# SCI Application Map Source of Truth

SCI is a commerce operating system, not a single application.

iTredVD is the SCI Commerce Console and vendor visibility engine. It must manage, monitor, and support all SCI commerce applications.

## SCI Ecosystem Map

SCI Commerce Operating System
- iTredVD / SCI Commerce Console
- SCI POS
- SCI Delivery
- SCI Marketplace
- SCI Access Hub
- SCI BI
- SCI Licensing
- SCI Activation
- Future SCI commerce applications

## iTredVD Role

iTredVD must act as the central control tower for the SCI ecosystem.

It must support:

- Vendor visibility
- Vendor onboarding
- Product intake
- Catalogue publishing
- Storefront publishing
- Subscription management
- License activation
- Pricing plan control
- Console staff management
- RPN and field operations
- WhatsApp Access Hub monitoring
- Application health monitoring
- Revenue reporting
- Audit logs
- Read diagnostics
- BI and analytics dashboards

## Rule

Future SCI applications must be integrated into iTredVD as monitored ecosystem applications, not as unrelated standalone systems.
