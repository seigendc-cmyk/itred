# UI and Brand Rules Source of Truth

SCI and iTredVD must use professional industrial-grade interface design.

Brand direction:

- iTred Marketplace
- Powered by seiGEN Commerce
- seiGEN Commerce Orange
- Charcoal matte grey
- Light professional backgrounds

UI rules:

- Sharp/square cards and panels
- No unnecessary rounded corners
- Mobile-first navigation
- Clean dashboard layouts
- Strong operational visibility
- Clear admin menus
- Minimal clutter
- Professional console behavior

The app must feel like a serious commerce operating system, not a casual marketplace template.
